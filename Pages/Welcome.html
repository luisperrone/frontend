Carrier Flags

fedex_save_cost_breakdown

What it does

Saves the detailed cost breakdown (including shipping zone) that is returned from the FedEx carrier response.

UI / System Impact
  • Location: SF Shipments Report
  • Without Flag: The 'Zone' column for FedEx shipments shows 'NULL'.
  • With Flag: The 'Zone' column correctly displays the shipping zone (e.g., "Zone 2", "Zone 8").

Shipping Flags

appsync_app

What it does

Switches the communication system for workstation printing between the modern (AppSync) and legacy (Pusher) methods.

UI / System Impact
  • With Flag (Enabled): Uses the modern **AppSync** system. This is faster and more secure.
  • Without Flag (Disabled): Uses the legacy **Pusher** system, which is older but supports all workstation app versions.

CRITICAL: When the flag is enabled, printing will fail silently if the workstation is not properly configured (e.g., missing a `computer_id`). The user will click print, nothing will happen, and no error message will be shown.
